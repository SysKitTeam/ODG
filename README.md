# Office 365 Data Generator

ODG is an open-source tool developed by [SysKit Ltd.](https://www.syskit.com/) to generate mock data for Office 365.

## ODG can generate the following Office 365 demo and sample data: 
* Users 
* Office 365 Groups
* Microsoft Teams
